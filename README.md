# 1 项目介绍
基于SSM+Vue的自习室预约系统：后端 SpringBoot、Mybatis-Plus，前端Vue+ElementUI，具体功能如下：
## 管理端
- 个人中心：查看和修改个人信息、修改密码
- 学生管理
- 座位预定管理
- 公告管理
- 自习室管理
- 留言板管理
- 系统管理(客服管理、轮播图管理)
## 用户端
- 个人中心：查看和修改个人信息、修改密码
- 座位预定管理
- 留言板管理
- 前台功能(查看公告、查看自习室、座位预定等)
